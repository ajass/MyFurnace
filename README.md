# MyFurnace Project

This repository contains documentation and information related to furnace and HVAC system evaluation, proposals, and requests for quotes.

## Contents

- `Furnace_details.md` - Technical specifications of existing furnace and heat pump equipment
- `curtis@atempheating.com_Here's Your Comfort Proposal.eml` - Original email proposal from A-TEMP Heating
- `curtis@atempheating.com_Here's Your Comfort Proposal.md` - Markdown conversion of the email proposal
- `Proposal - Denis Roberts 07-31-2025 8 30 AM-1.pdf` - PDF proposal from A-TEMP Heating
- `Proposal - Denis Roberts 07-31-2025 8 30 AM-1.md` - Markdown conversion of the PDF proposal

## Purpose

This project is used to:
1. Document existing HVAC equipment specifications
2. Store and review contractor proposals
3. Prepare requests for quotes (RFQs) for HVAC work
4. Track HVAC system evaluations and upgrade options

## Equipment Details

- **Furnace**: Goodman CAPF3137B6AA
- **Heat Pump/AC**: Goodman GSX140301NC (R-410A refrigerant)

## Usage

The markdown files can be viewed directly in GitHub or downloaded for local review. The original email and PDF formats are also preserved for reference.
